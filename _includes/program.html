Android

Com mais de dois mil milhões de utilizadores activos, o Android é o principal Sistema Operativo móvel. Não estando apenas restrito aos telemóveis, neste momento é possível interagir com o Android em TV's, no carro, nos relógios, etc.

Firebase

Encontrando-se disponível tanto para Android como para iOS, o Firebase permite enriquecer as aplicações móveis com inúmeras novas funcionalidades: autenticação, base de dados remota, notificações push, crashlytics, etc.

TensorFlow

Duas das palavras mais faladas no mundo tecnológico neste momento são: inteligência artificial. Inúmeras aplicações estão a tirar partido do TensorFlow para adicionarem novas funcionalidades às suas aplicações.
